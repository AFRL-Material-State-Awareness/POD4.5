get_call {insight}R Documentation

Get the model's function call

Description

Returns the model's function call when available.

Usage

get_call(x)

Arguments

x

A fitted mixed model.

Value

A function call.

Examples

data(mtcars)
m <- lm(mpg ~ wt + cyl + vs, data = mtcars)
get_call(m)

if (require("lme4")) {
  m <- lmer(Sepal.Length ~ Sepal.Width + (1 | Species), data = iris)
  get_call(m)
}
