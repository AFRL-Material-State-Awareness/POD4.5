Generate random sample from a discrete uniform distribution

Description

Generate random sample from a discrete uniform distribution

Usage

rdunif(n, b, a = 1)

Arguments

n

Number of samples to draw.

a, b

Range of the distribution (inclusive).

Examples

table(rdunif(1e3, 10))
table(rdunif(1e3, 10, -5))
